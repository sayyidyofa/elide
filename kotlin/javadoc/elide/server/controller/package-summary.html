  • Class Summary 
    Class Description
    BaseControllerAbstract base implementation of built-in Elide controller abstracts.
    AssetReferenceBuilderContext handler to collect asset configuration.
    PageControllerDefines the built-in concept of a Page-type handler, which is capable of performing SSR, serving static assets, and handling page-level RPC calls.
    PageWithPropsExtends PageController with support for page-level State, produced via the props method; computed state is shared with VM render executions, and can additionally be injected into the page for use by frontend code (typically to hydrate a server-rendered UI).
