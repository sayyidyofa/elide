ProtoBufGeneratorOptions

public final class ProtoBufGeneratorOptions

Protobuf Generator: Options

Specifies the set of options which apply to the protocol buffer source generator. The generator can generate Proto2 or Proto3 syntax from regular Kotlin models, with special annotations for controlling the output.

Parameters

packageName

package name to use for generated models; if null, a sensible one will be generated.

packageOptions

top-level package options to apply.

protoOptions

proto options to apply.

emitWarningComments

whether to emit file warnings indicating generated code.

syntaxVersion

protocol buffers syntax version to emit.
