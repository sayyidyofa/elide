public final class Deprecated extends ProtoOption.KnownFileOption<Boolean>

File option: deprecated

Marks an entire .proto file full of structures/enums, etc., as deprecated. In applicable languages and target platforms, this generates corresponding deprecation signals or warnings.

From the Protocol Buffers documentation:

Is this file deprecated? Depending on the target platform, this can emit Deprecated annotations for everything in the file, or it will be completely ignored; in the very least, this is a formalization for deprecating files
