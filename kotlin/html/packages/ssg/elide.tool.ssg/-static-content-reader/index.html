public interface StaticContentReader

SSG: Static Content Reader

Defines an interface for an object which is responsible for (1) converting HTTP responses to persisted string content for a given SSG handler, and (2) parsing such content for references to discovered URLs which should be made eligible for further processing.

The static content reader is used by the AppStaticCompiler after a request has been resolved to an HTTP response. Typically, the string body of the response is persisted to disk as the result.

Some HTTP references in the resulting body may be eligible for further processing, so these are parsed as HTML (where applicable), and then handed up to the SiteCompiler for further action.
