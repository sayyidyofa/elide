compileStaticSiteAsync

public abstract Deferred<SiteCompileResult> compileStaticSiteAsync(Integer count, LoadedAppInfo appInfo, Sequence<StaticFragmentSpec> seed, StaticSiteBuffer buffer)

Run the static site compiler with the provided expected seed inputs (of length count); if so directed via the params passed to prepare, the compiler will also attempt to discover additional URLs in returned HTML responses.

This method will execute until all requests have been fulfilled, or until the compiler has been interrupted. A call is considered "fulfilled" when the following conditions are true:

  1. A response has been received from the application, or

  2. An error has been encountered while fulfilling the request, or

  3. The request timeout value elapsed (a subset of case 2).

This method operates asynchronously, by returning a Deferred job which evaluates to the compiler result. For synchronous dispatch, see compileStaticSite.

Return

Compiler execution result.

See also

AppStaticCompiler.compileStaticSite

for synchronous dispatch.

Parameters

count

Count of seed requests to begin the compilation with.

appInfo

Loaded application manifest and info.

seed

Expected requests which should be used to start the execution.

buffer

Buffer which holds the results from each request/response cycle.

Throws

SSGCompilerError

in the event of an unrecoverable error.
